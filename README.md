# Cloud Engineer Roadmap

This repository is my structured learning roadmap for becoming a cloud engineer. I built it to document my hands-on progress across core cloud engineering domains, including compute, databases, DevOps, Kubernetes, networking, security, Terraform, and machine learning.

## Why I Built This Repository

I created this roadmap so recruiters, hiring managers, and technical reviewers can quickly see:

- the cloud topics I am learning
- the services and tools I am practicing
- how I organize technical knowledge
- the areas I am building toward for real cloud engineering work

This repo is meant to show more than a list of skills. It is a working structure for tracking practical growth, project notes, labs, and future cloud portfolio work.

## What This Repository Is For

Each folder represents a key area of cloud engineering. As I continue learning, I will add notes, diagrams, labs, scripts, Terraform configurations, and project documentation inside the relevant sections.

The goal is to make my learning visible, organized, and easy to review.
