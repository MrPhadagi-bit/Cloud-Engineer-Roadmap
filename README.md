# Cloud Engineer Roadmap

> **A structured, hands-on learning portfolio for cloud engineering  tracking practical progress across AWS infrastructure, DevOps automation, container orchestration, and applied ML services.**

---

## About This Repository

This repository is my living portfolio for becoming a cloud engineer. I built it to document real, hands-on progress across core cloud domains — not just list skills, but show how I think about infrastructure, solve problems, and build toward production-ready systems.

**What you'll find here:**
- **Topic-specific learning tracks** — organized by cloud domain (Compute, Networking, Security, DevOps, etc.)
- **Hands-on labs & configurations** — Terraform modules, Kubernetes manifests, CI/CD pipelines, and security policies
- **Architecture notes & diagrams** — documenting design decisions and trade-offs
- **Project documentation** — real-world scenarios, not tutorial walkthroughs
